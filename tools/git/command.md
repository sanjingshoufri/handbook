# 指令
## 配置
- git config --list       //查看所有的配置
- git config key          //查看某个key的值
- git help config         //查看config的帮助

## 库
- git clone             //克隆一个ref源
    - 使用SSH和HTTPS方式的不同

## version
- git checkout     //切换分支或者恢复工作区的文件
    - git checkout -b|-B <new_branch>;   //创建一个新的分支

- git fetch        //    

- git reset        //重置当前的HEAD为指定状态
