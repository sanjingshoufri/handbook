# 编程语言基础
- 静态和动态的区别
	- 静态语言
		- 允许编译器去决策问题
			- 比如java、C++
	- 动态语言
		- 当项目执行时，才能决策问题
			- 比如PHP、Python

- 声明的范围
	- 如果很容易判断声明的范围，那么语言使用静态范围或词汇范围，反之使用动态范围.

- 环境和状态
	- 在存储里，环境是一个从名字到地址的映射.
	- 情况是一个从地址到值的映射.

- 静态范围和块结构
	- 声明的范围是由声明的地点而决定的.
	- 状态是从地址到值的映射.

- 显示访问控制
- 动态范围
	- 声明
		- 声明告诉我们某些事情的类型
	- 定义
		- 定义告诉我们某些事情的值
- 参数传递机制
	- 实参与形参是如何通信的
		- 值调用
			- 如果实参是变量，则拷贝变量;如果实参是表达式，则求和. 
			- 值放在调用的存储过程相应形参的位置
			- 
		- 引用调用
		- 名调用
- 别名
	- 两个相同的正式参数会引用相同的地方，这样称为别名.
